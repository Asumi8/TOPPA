# README

## 開発言語
- Ruby 3.0.1
- Rails 6.1.7

## 就業Termの技術
- devise
- Ajaxを使ったコメント機能
- AWS EC2へのデプロイ

## カリキュラム外の技術
- gem/chartkick
- gem/devise_invitable
- gem/whenever
- gem/cancancan
- gem/rails_admin

## 実行手順
````
$ git clone https://github.com/Asumi8/TOPPA.git
$ cd TOPPA
$ bundle install
$ rails db:create && rails db:migrate
$ rails s
````
